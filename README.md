StudyAssist is a modern, fully responsive, multi-page web application developed using **React** to provide professional **Statement of Purpose (SOP)** and **Letter of Recommendation (LOR)** writing services for students aspiring to study abroad.

The platform explains services clearly, offers country-specific guidance, displays pricing packages, and includes a **fully functional contact form with real email delivery**, making it production-ready and deployment-ready.



Project Overview

Students applying to international universities often struggle with:
- Writing strong SOPs and LORs
- Understanding country-specific admission requirements
- Finding reliable and professional writing assistance

**StudyAssist** solves this problem by providing a centralized, easy-to-use, and professional web platform that connects students with SOP/LOR writing experts.



Objectives

- Build a modern multi-page website using React
- Clearly explain SOP & LOR writing services
- Provide country-specific study abroad guidance
- Display transparent pricing packages
- Implement a working contact form with email delivery
- Ensure full mobile responsiveness
- Prepare the application for deployment on Vercel or GitHub Pages
